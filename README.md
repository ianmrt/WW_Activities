
# Adventurous Activities Dashboard

## Deployment

1. Create a GitHub repository.
2. Upload:
   - app.py
   - requirements.txt
   - data.xlsx
3. Push to GitHub.
4. Sign in to Streamlit Community Cloud.
5. Create a new app from the repository.
6. Select app.py as the entry point.

## Updating Data

Replace data.xlsx with the latest workbook and commit the change.

The dashboard automatically refreshes within a few minutes.
